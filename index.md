# Clawland

**Build the Edge AI Future. Get Paid for It.**

Clawland is an open-source ecosystem of AI Agents that run on hardware from $2 microcontrollers to cloud servers — replacing human monitoring, maintenance, and service jobs at 1/100th the cost.

## Quick Start

### Choose Your Agent

| If you have... | Use... | Get started |
|----------------|--------|-------------|
| $10 RISC-V board (LicheeRV-Nano) | **PicClaw** | [picclaw docs →](https://github.com/Clawland-AI/picclaw) |
| $50 Raspberry Pi | **NanoClaw** | [nanoclaw docs →](https://github.com/Clawland-AI/nanoclaw) |
| $2 ESP32 MCU | **MicroClaw** | [microclaw docs →](https://github.com/Clawland-AI/microclaw) |
| Cloud server | **MoltClaw** | [moltclaw docs →](https://github.com/Clawland-AI/moltclaw) |

### Build a Sensor Kit

Pick a [ready-made kit](https://github.com/Clawland-AI/clawland-kits) with full BOM, wiring diagram, and pre-configured skills.
